目前被广泛使用的RxJS版本有两个：v4和v5，这两个版本的API很相似，但是也有巨大的差别。

如无特殊说明，所有代码都按照v5来讲解，如果涉及v4和v5的API差别，也会单独标注。

### npm 安装


```javascript
// v5
npm install rxjs
// v4
npm install rx
```
